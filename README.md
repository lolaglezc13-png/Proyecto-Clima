# 🌦️ Proyecto: Vigilante Climático AEMET

## 📝 Descripción
Este proyecto es un sistema automático desarrollado en **Python** que se conecta a la API de **AEMET (Agencia Estatal de Meteorología)** para monitorizar el clima en tiempo real. El objetivo es recopilar datos, analizarlos y generar alertas automáticas si se detectan temperaturas extremas.

## 🚀 Funcionalidades
- **Extracción Automática:** Conexión con la API de AEMET usando una clave VIP.
- **Análisis de Datos:** Cálculo de temperaturas máximas y mínimas mediante la librería **Pandas**.
- **Visualización:** Generación de gráficos de barras con **Matplotlib** resaltando los extremos en colores.
- **Sistema de Alerta:** Detección de olas de calor y generación de informes de incidencias.
- **Histórico:** Guardado de datos con fecha automática para crear una base de datos histórica.

## 🛠️ Herramientas utilizadas
- **Lenguaje:** Python 3
- **Librerías:** - `Requests`: Para navegar por internet.
  - `Pandas`: Para organizar los datos en tablas.
  - `Matplotlib`: Para crear los gráficos.
  - `Datetime`: Para gestionar el calendario y los nombres de archivos.
